# Упражнение для начинающих Python-разработчиков

Это простое упражнение поможет вам познакомиться с основами Python и научиться проходить автоматические тесты с использованием `pytest`.

## Цель

Ваша задача — отредактировать файл `main.py`, чтобы он выводил вашу личную информацию, и затем запустить тесты, чтобы убедиться, что все сделано правильно.

## Инструкция

### Шаг 1: Установите `Виртуальное окружение`

Выполните команду:

```bash
python -m venv venv
```
Затем активируйте его:

```bash
source venv/Scripts/activate
```

И установите зависимости в проект:
```bash
pip install -r requirements.txt
```

### Шаг 2: Отредактируйте файл `main.py`

1.  Откройте файл `main.py` в вашем любимом редакторе кода.
2.  Замените текст-заполнитель `[вставьте сюда свое имя]` на ваше настоящее имя.
3.  Замените `[вставьте сюда свой возраст]` на ваш возраст.
4.  Замените `[вставьте сюда свой любимый цвет]` на ваш любимый цвет.

**Пример того, как может выглядеть измененный файл:**

```python
# ...
print("Меня зовут: Алексей")
print("Мой возраст: 30")
print("Мой любимый цвет: синий")
# ...
```

### Шаг 3: Запустите тесты

После того как вы сохранили изменения в `main.py`, вам нужно запустить тесты.

1.  Откройте терминал или командную строку.
2.  Убедитесь, что вы находитесь в той же директории, что и файлы `main.py` и `test_main.py`.
3.  Выполните следующую команду:

```bash
pytest
```

### Что должно произойти?

*   **Если вы все сделали правильно**, вы увидите сообщение об успешном прохождении всех тестов. Оно будет выглядеть примерно так:

```
============================= test session starts ==============================
...
collected 6 items

test_main.py ......                                                      [100%]

============================== 6 passed in ...s ===============================
```

*   **Если что-то пошло не так**, `pytest` покажет, какие именно тесты упали, и даст подробную информацию об ошибке. Например, если вы забыли вставить свое имя:

```

```
Это сообщение гораздо информативнее и сразу указывает на проблему.

В этом случае вернитесь к Шагу 2 и убедитесь, что вы правильно отредактировали и сохранили файл `main.py`.

Удачи!
